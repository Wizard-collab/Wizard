Wizard is built with Python 3.7.3 and PyQt 5.11.3 

All the Python dependencies are included in the install ( compiled )

They include :

	- `Python 3.7.3 <https://www.python.org/downloads/release/python-373/>`_ :
		
		- PyQt5
		- PyYaml
		- psutil
		- playsound
		- pyperclip
		- watchdog
		- google-api-python-client
		- lxml
		- pillow
		- opencv-python
		- pyautogui
		- DesktopMagic
		- pywin32
		- QScintilla
		- markdown

	- Python 2.7.17 ( Only sites-packages ) :

		- mss
		- PIL
		- PyQt5 ( Non official )
		- watchdog
		- yaml
